About Me

Cho Hyeonjun

안녕하세요! 저는 UX/UI 디자인과 웹 퍼블리싱을 하는 3년 차 웹 퍼블리셔 조현준입니다. 웹 환경에서의 디자인은 사용자를 의도된 목적지로 편안하게 유도하는 안내자와 같다고 생각합니다. 웹 표준과 웹 접근성을 준수한 마크업, 우선순위를 고려한 구조로 자연스러운 흐름을 만듭니다.

Photoshop, Illerstrater 등을 이용해 웹사이트의 레이아웃과 디자인을 구성하고, HTML, CSS, JavaScript 등을 활용하여 시안을 실제 웹사이트에 구현합니다.

무엇보다 사용자가 납득하고 편리하게 이용할 수 있는 웹 환경을 추구하지만, 미적 트렌드와 개발 기술에도 관심이 많으며 새로운 도전에 항상 열려 있습니다.
